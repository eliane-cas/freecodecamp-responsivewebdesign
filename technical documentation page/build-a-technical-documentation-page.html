Understanding the Code

Learn what each html function is doing. The code you will use to create your calculator is made up of many pieces of syntax that work together to define different elements of a document.

  • html This piece of syntax tells the rest of the document what language is being used in the code.
  • head Tells the document that everything beneath it is data about data, also known as "metadata".
  • title This is where you will name the title of your document. This attribute is used to define what the title of the document will be when opened in an html browser.
  • body bgcolor= "#" This attribute sets the color of the code's background and body.
  • text= "" The word in this set of quotes sets the color of the text on the document.
  • form name="" This attribute specifies the name of a form, which is used to build the structure of what comes after it based on what Javascript knows that form name to mean.
  • input type=""> This is where the action happens. The "input type" attribute tells the document what sort of text the values in the rest of the brackets are.
  • value="" This command tells the document what will be contained in the input type specified above.
  • onClick="" This syntax describes an event, which tells the document that something should occur when the button is clicked.
  • br this tag initiates a line break in the document, so that whatever comes after it will appear a line below whatever came before it.

Creating the HTML File

Open a text editing program on your computer. There are a number of programs you can use, but for convenience and quality, we recommend using TextEdit or Notepad.

Paste the HTML code for a calculator into the document.

Add an HTML extension to the file name. In the "Save As..." menu, type in your file name followed by ".html", and then click "Save".

Using Your Calculator

Find your file to open it and click on it.

Extra Stuff

There are plenty of ways to do math on a desktop computer using a built-in calculator, but another way is to build one yourself using a simple HTML code. To create a calculator using HTML, learn some basics about HTML, then copy the necessary code into a text editor and save it with an HTML extension. You can then use your calculator by opening up the HTML document in your favorite browser. By doing all of this, not only will you be able to do math in a browser, but you can also learn some fundamentals about the art of coding!
